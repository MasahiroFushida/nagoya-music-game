授業概要

この授業で何を学ぶか

ゲーム1とは

Unityでゲームを作る授業です

  • 対象：プログラミング経験ゼロでもOK
  • 使うツール：Unity（無料）+ Visual Studio Code
  • 言語：C#（シーシャープ）
  • 目標：自分で企画したゲームを完成させて発表する

🎵 音楽大学の強みを活かして、「音」や「音楽」を使ったゲームも大歓迎です！

なぜゲーム制作を学ぶのか

ゲームと音楽は切り離せない

  • ゲームサウンドの市場は年々拡大中（BGM・SE・ボイス）
  • インタラクティブな音楽表現の手段としてのゲームエンジン
  • 音楽×プログラミングで生まれるオリジナルな表現
  • 就職・フリーランス・個人制作など幅広い活用の場

プログラミングは「楽器」と同じ。
最初は難しくても、練習すれば必ず弾けるようになります。

授業のルール

  • 毎回PCを持参してください（必須）
  • 授業中は授業に関係のないアプリを閉じる
  • 分からないことはその場で質問してOK
  • 作業が早く終わった人は自由に発展してOK
  • AIツール（ChatGPT等）の活用はOK、ただし中身を理解すること

「完璧なゲームを作ること」より「少しずつ理解しながら進めること」が大切です。

PC環境の確認

Unity に必要な PC スペック

🪟 Windows

  • OS：Windows 10 / 11（64bit）
  • CPU：Intel Core i5 / AMD Ryzen 5 以上
  • メモリ：8GB以上（16GB推奨）
  • GPU：DirectX 11 対応
  • ストレージ：空き 10GB 以上

🍎 Mac

  • OS：macOS 12（Monterey）以上
  • CPU：Intel Core i5 / Apple Silicon（M1〜）
  • メモリ：8GB以上（16GB推奨）
  • GPU：Metal 対応（対応Mac はすべてOK）
  • ストレージ：空き 10GB 以上
⚠️ メモリ 4GB 以下のPCは動作が不安定になる場合があります。事前に確認してください。

C#（シーシャープ）

C# とは何か

  • Microsoft が開発したプログラミング言語
  • Unity のスクリプトで使う唯一の言語
  • 「変数・条件分岐・繰り返し」など基礎から学んでいく
  • ゲーム以外にもアプリ・Webサービス開発にも使われる汎用言語
  • 最初は難しく感じるが、パターンを覚えれば書けるようになる

セットアップ

開発環境を整えよう

インストール手順

Unity のインストール

1
Unity Hub をダウンロード unity.com/ja/download を開いて「Unity Hub をダウンロード」を選択
2
Unity Hub をインストール・起動 ダウンロードしたインストーラーを実行して起動
3
Unity アカウントを作成（または既存アカウントでサインイン） id.unity.com で無料の Personal ライセンスを作成
4
Unity Editor をインストール Unity Hub → 「インストール」→ 「エディターをインストール」→ 2022.3.62f3（LTS）を選択 → 「Visual Studio」も一緒にチェック
インストール時の注意

つまずきやすいポイント

  • インストール先のパスに日本語・スペースを含めない（例：C:\Unity など）
  • ストレージの空き容量は最低 10GB 以上必要
  • インストールには時間がかかります（20〜60分）
  • 途中でエラーが出たら遠慮なく呼んでください
⚠️ もし今日インストールが終わらない場合は、次回までに自宅で完了させてきてください。
動作確認

インストール確認

1
Unity Hub を起動する
2
「新しいプロジェクト」をクリック テンプレートは「3D（コア）」を選択
3
プロジェクト名を入力して「プロジェクトを作成」 名前は何でもOK（例：MyFirstGame）
4
Unityのエディター画面が起動すれば成功！ Scene ビューに広場のようなものが見えればOK
Visual Studio Code

VS Code とは何か

  • Microsoft が開発した無料のコードエディター
  • 軽量で起動が速く、拡張機能で機能を追加できる
  • Unity のスクリプト（C#）を書くために使う
  • コードの色分け・補完・エラー表示などが使えて便利
  • 世界中の開発者が使っており、情報が豊富

公式サイト：code.visualstudio.com
ダウンロード：code.visualstudio.com/download

Visual Studio Code

VS Code のセットアップ

  • 1. C# 拡張機能をインストール
    VS Code 左サイドバーの拡張機能アイコン → 「C#」で検索 → インストール
  • 2. Unity Editor から VS Code を設定
    Unity Editor → メニュー「Edit」→「Preferences」（Mac: Unity → Settings）
    → 「External Tools」セクション → 「External Script Editor」から VS Code を選択
  • 3. 動作確認
    Create Asset → C# Script でスクリプト作成 → ダブルクリックで VS Code で開く
