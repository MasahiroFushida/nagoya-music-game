授業概要

この授業で何を学ぶか

ゲーム1とは

Unityでゲームを作る授業です

  • 対象：プログラミング経験ゼロでもOK
  • 使うツール：Unity（無料）+ Visual Studio Code
  • 言語：C#（シーシャープ）
  • 目標：自分で企画したゲームを完成させて発表する

🎵 音楽大学の強みを活かして、「音」や「音楽」を使ったゲームも大歓迎です！

なぜゲーム制作を学ぶのか

ゲームと音楽は切り離せない

  • ゲームサウンドの市場は年々拡大中（BGM・SE・ボイス）
  • インタラクティブな音楽表現の手段としてのゲームエンジン
  • 音楽×プログラミングで生まれるオリジナルな表現
  • 就職・フリーランス・個人制作など幅広い活用の場

プログラミングは「楽器」と同じ。
最初は難しくても、練習すれば必ず弾けるようになります。

授業のルール

  • 毎回PCを持参してください（必須）
  • 授業中は授業に関係のないアプリを閉じる
  • 分からないことはその場で質問してOK

「完璧なゲームを作ること」より「少しずつ理解しながら進めること」が大切です。

インストール手順

Unity のインストール

1
Unity Hub をダウンロード unity.com → 「Unity をダウンロード」→「Unity Hub」を選択
2
Unity Hub をインストール・起動 ダウンロードしたインストーラーを実行して起動
3
Unity アカウントを作成（または既存アカウントでサインイン） 無料の Personal ライセンスでOK
4
Unity Editor をインストール Unity Hub → 「インストール」→ LTS 版（推奨版）を選択 → 「Visual Studio」も一緒にチェック
インストール時の注意

つまずきやすいポイント

  • インストール先のパスに日本語・スペースを含めない（例：C:\Unity など）
  • ストレージの空き容量は最低 10GB 以上必要
  • インストールには時間がかかります（20〜60分）
  • 途中でエラーが出たら遠慮なく呼んでください
⚠️ もし今日インストールが終わらない場合は、次回までに自宅で完了させてきてください。
動作確認

インストール確認

1
Unity Hub を起動する
2
「新しいプロジェクト」をクリック テンプレートは「3D（コア）」を選択
3
プロジェクト名を入力して「プロジェクトを作成」 名前は何でもOK（例：MyFirstGame）
4
Unityのエディター画面が起動すれば成功！ Scene ビューに広場のようなものが見えればOK
