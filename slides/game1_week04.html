Section 01

Translateで毎フレーム動かす

Vector3 の6方向を使って移動を表現する

Test1-4

Update で毎フレーム前進させる

void Start() { Debug.Log("Start"); } void Update() { Debug.Log("Update"); transform.Translate(Vector3.forward); //transform.Translate(Vector3.forward * 0.1f); }

Start は1回だけ、Update は毎フレーム呼ばれる。Console を見ると Update が大量に出力されることがわかる。

Vector3 の方向

6つの方向ベクトル

  • Vector3.forward ＝ Z軸の正方向 (0, 0, 1)
  • Vector3.back ＝ Z軸の負方向 (0, 0, -1)
  • Vector3.right ＝ X軸の正方向 (1, 0, 0)
  • Vector3.left ＝ X軸の負方向 (-1, 0, 0)
  • Vector3.up ＝ Y軸の正方向 (0, 1, 0)
  • Vector3.down ＝ Y軸の負方向 (0, -1, 0)

方向ベクトルに数値を掛けると速さを調整できる：Vector3.forward * 0.1f

Section 02

public変数とif文

Inspectorから値を変える／条件で処理を分岐する

Test2-1

速度を public 変数にする

public float speed; void Update() { transform.Translate(Vector3.forward * speed); if (transform.position.z >= 30) { Debug.Log("30以上"); //speed = -0.1f; //speed *= -1; //speed = -speed; } }

public float speed; と書くと Inspector に「Speed」欄が出てきて、ゲームを止めずに速さを調整できる。

if 文と比較演算子

条件で処理を分岐する

  • < ：より小さい
  • > ：より大きい
  • <= ：以下
  • >= ：以上
  • == ：等しい ／ != ：等しくない
// 位置 z が 30 以上になったら if (transform.position.z >= 30) { Debug.Log("30以上"); }
Test2-2

Destroy でオブジェクトを消す

public float speed; public int destroyPoint; void Update() { transform.Translate(Vector3.forward * speed); if (transform.position.z >= destroyPoint) { //このオブジェクトを消す Destroy(gameObject); } }
  • Destroy(gameObject)：自分自身（このスクリプトが付いているオブジェクト）を削除
  • 消滅地点の数値も public int destroyPoint; として Inspector で調整できる
Section 03

Rigidbodyで物理演算

力を加えて自然な動きを作る

Test2-3

Rigidbodyを取得して力を加える

Rigidbody rb; // Rigidbodyを入れるための変数 // 変数の型 変数の名前 void Start() { rb = GetComponent<Rigidbody>(); // 変数の名前 = GetComponent<型>(); rb.AddForce(Vector3.forward * 1000); // 変数の名前.AddForce(力の方向); }

Sphere に Rigidbody コンポーネントを付けてから実行すると、Start の瞬間に前方へ強く飛んでいく。

AddForce

Translate との違い

transform.Translate

位置を直接動かす。
重力や衝突を無視。
毎フレーム書く必要あり。

Rigidbody.AddForce

力を加えて物理で動かす。
重力・慣性・衝突が働く。
1回加えれば惰性で進み続ける。

⚠ Rigidbody が無いオブジェクトに rb.AddForce を呼ぶとエラーになる。Inspector から Rigidbody を必ず追加しよう。
Section 04

キー入力で操作する

矢印キー＋スペースで Sphere を動かす

Test2-4 ①

矢印キーで4方向に動かす

void Update() { // 十字キーで動かす if (Input.GetKey(KeyCode.RightArrow)) rb.AddForce(Vector3.right); // 右に力を加える if (Input.GetKey(KeyCode.LeftArrow)) rb.AddForce(Vector3.left); // 左に力を加える if (Input.GetKey(KeyCode.UpArrow)) rb.AddForce(Vector3.forward); // 前に力を加える if (Input.GetKey(KeyCode.DownArrow)) rb.AddForce(Vector3.back); // 後ろに力を加える }

Input.GetKey：キーを押している間ずっと true ＝ 押し続けるとどんどん加速する。

Test2-4 ②

スペースキーでジャンプ

if (Input.GetKeyDown(KeyCode.Space)) { rb.AddForce(Vector3.up * 200); // 上に力を加える }
  • GetKeyDown：押した瞬間だけ true（ジャンプはこちら）
  • GetKey：押している間ずっと true（移動はこちら）
  • 使い分けないと、押しっぱなしで無限に飛んでしまう
  • Vector3.up * 200：重力に勝つ大きな値が必要（数値を変えて高さを調整）
