Section 01

Transformを使って動かす

スクリプトから位置・回転・スケールを制御する

Transform の復習

position / rotation / scale

  • position（位置）：Vector3 型の座標 (x, y, z)
  • rotation（回転）：Quaternion 型（内部的に4つの数値）
  • scale（拡縮）：Vector3 型のサイズ倍率
  • スクリプトからは transform という変数名でアクセスできる

Vector3 とは

X・Y・Z の3つの値をまとめた型。
new Vector3(1f, 0f, 0f) のように書く

World vs Local

transform.position はワールド座標。
transform.localPosition は親からの相対座標

Transform 操作

スクリプトから Transform を変える

// 位置を直接セットする transform.position = new Vector3(0f, 1f, 0f); // 現在の位置から少しずらす transform.position += new Vector3(0.1f, 0f, 0f); // Translate を使って相対移動（よく使う） transform.Translate(0.1f, 0f, 0f); // 回転を加える transform.Rotate(0f, 1f, 0f);

transform.Translate(x, y, z) は「今いる位置からの相対移動」なので直感的に使いやすい。

Update メソッド

毎フレーム呼ばれる処理

  • Playモード中、Unity は1秒間に約60回（フレーム）ループする
  • Update() はそのたびに呼ばれる ＝ 毎秒約60回実行される
  • 「少しずつ動かす」処理を Update に書くとなめらかなアニメーションになる
  • Start は初期化のみ、動き続ける処理は Update に書くのが基本

Update() の中で毎フレーム「0.01 だけ移動」を繰り返すと、オブジェクトがスムーズに動き続けます。

コード例

毎フレーム少しずつ右に移動する

public class MoveObject : MonoBehaviour { void Update() { // 毎フレーム X 方向に 0.05 移動 transform.Translate(0.05f, 0f, 0f); } }
  • このままだと画面外に飛んでいってしまう
  • 次はキー入力と組み合わせて「押している間だけ動く」ようにする
⚠ フレームレートは環境によって変わる。後で Time.deltaTime を使って補正します。
Section 02

キーボード入力を受け取る

Input クラスを使ったキー入力処理

Input.GetKey

キーが押されているか判定する

// キーを押している間ずっと true if (Input.GetKey(KeyCode.RightArrow)) { // 右矢印キーを押しているとき } // キーを押した瞬間だけ true if (Input.GetKeyDown(KeyCode.Space)) { // スペースキーを押した瞬間 } // キーを離した瞬間だけ true if (Input.GetKeyUp(KeyCode.RightArrow)) { // 右矢印キーを離した瞬間 }
コード例

上下左右キーで Cube を動かす

public class PlayerMove : MonoBehaviour { void Update() { if (Input.GetKey(KeyCode.RightArrow)) transform.Translate( 0.05f, 0f, 0f); if (Input.GetKey(KeyCode.LeftArrow)) transform.Translate(-0.05f, 0f, 0f); if (Input.GetKey(KeyCode.UpArrow)) transform.Translate(0f, 0.05f, 0f); if (Input.GetKey(KeyCode.DownArrow)) transform.Translate(0f, -0.05f, 0f); } }
Time.deltaTime

フレームレートに依存しない移動

  • フレームレートが高い PC では Update が多く呼ばれ → 速く動きすぎる
  • フレームレートが低い PC では遅くなる ＝ 環境によって速さが変わる
  • Time.deltaTime：前フレームからの経過時間（秒）。高FPS時は小さく、低FPS時は大きい
  • 移動量に掛け算することで「1秒あたりの移動距離」が一定になる
// speed = 3 なら 1秒で 3ユニット移動（FPS に関係なく一定） transform.Translate(3f * Time.deltaTime, 0f, 0f);

Playモードと Inspector での調整

動作確認

▶ ボタンで Playモード開始Game ビューがアクティブになる

4

public 変数

Inspector からスピードを調整する

public class PlayerMove : MonoBehaviour { // public にすると Inspector で値を変えられる public float speed = 3f; void Update() { if (Input.GetKey(KeyCode.RightArrow)) transform.Translate(speed * Time.deltaTime, 0f, 0f); if (Input.GetKey(KeyCode.LeftArrow)) transform.Translate(-speed * Time.deltaTime, 0f, 0f); if (Input.GetKey(KeyCode.UpArrow)) transform.Translate(0f, speed * Time.deltaTime, 0f); if (Input.GetKey(KeyCode.DownArrow)) transform.Translate(0f, -speed * Time.deltaTime, 0f); } }

Inspector の「Speed」フィールドを変えるだけでコードを書き直さずにスピード調整できる！
