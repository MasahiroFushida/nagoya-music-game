Section 01

Unityの画面を知ろう

エディターの各エリアと役割

エディター概要

5つのエリア

Scene ビュー

3D空間を自由に見回しながらオブジェクトを配置・編集する作業エリア

Game ビュー

実際にゲームを動かしたときのカメラ映像。Playボタンで確認する

Hierarchy

シーン内のオブジェクトが一覧表示される。親子関係もここで管理

Inspector

選択中のオブジェクトの詳細設定（位置・コンポーネントなど）を表示

＋ Project：プロジェクト内のファイル（アセット）を管理するブラウザ

Scene ビュー

オブジェクトを配置する作業台

  • マウスホイールでズームイン・アウト
  • 右ドラッグ（またはAlt+ドラッグ）で視点を回転
  • 中ボタンドラッグ（またはAlt+右ドラッグ）でパン（平行移動）
  • オブジェクトをクリックで選択 → 矢印ツールで移動
  • ツールバーの W・E・R で 移動・回転・スケール を切り替え
⚠ Scene ビューで動かしてもゲームは動きません。あくまで「配置の作業台」です。
Game ビュー

実際のゲーム画面を確認する

  • ▶（Play）ボタンを押すとゲームが始まり、Game ビューに映像が映る
  • カメラオブジェクトの向き・位置がそのまま映像になる
  • Playモード中でも一時停止（⏸）やコマ送り（⏭）ができる
  • 解像度・アスペクト比を上部ドロップダウンで変更可能

Game ビューは「プレイヤーが見る画面」のプレビューです。カメラを動かすと映像も変わります。

Hierarchy

シーン内オブジェクトの一覧

  • シーンに存在するすべてのオブジェクトが表示される
  • 左の▶アイコンで子オブジェクトを展開・折りたたみ
  • 右クリック → 各種メニューからオブジェクトを追加・削除
  • オブジェクトをドラッグして別のオブジェクトの下に入れると「子」になる
  • 親が動けば子も一緒に動く（親子関係の基本）

Hierarchy上のオブジェクトをクリックすると Scene ビューで選択状態になります。

Inspector

選択オブジェクトの詳細設定

  • Hierarchy でオブジェクトを選択すると Inspector に詳細が表示される
  • Transform：位置（Position）・回転（Rotation）・拡縮（Scale）
  • コンポーネント：Mesh Renderer、Rigidbody など、機能の追加単位
  • 「Add Component」ボタンで新しい機能を追加できる
  • 数値フィールドに直接入力、またはドラッグで変更

Unityでは「コンポーネントを組み合わせてオブジェクトの動作を決める」という考え方が基本です。

Project ウィンドウ

ファイル（アセット）の管理

  • プロジェクトフォルダ内のファイルをすべて表示するブラウザ
  • スクリプト / 画像 / 音声 / 3Dモデル などがここに集まる
  • 右クリック → Create でスクリプトやフォルダを作成
  • ファイルを Scene や Hierarchy にドラッグして使用する
  • フォルダで整理する習慣をつけると後が楽（Scripts / Materials / Prefabs など）
⚠ Project ウィンドウ外（Finderや Explorer）でファイルを移動すると参照が壊れます。必ずUnity上で操作しましょう。
Section 02

オブジェクトを置いてみよう

3D オブジェクトの追加と Transform 編集

オブジェクトの追加

Cubeをシーンに置く手順

1
Hierarchy を右クリックコンテキストメニューを開く
2
3D Object → Cube（Sphere / Plane / Cylinder なども同様）
3
Scene ビューに Cube が現れるHierarchy にも追加されていることを確認
4
オブジェクト名を変更するHierarchy 上でダブルクリックして分かりやすい名前に
Transform

位置・回転・スケールを変える

  • Position（位置）：X / Y / Z 軸の座標。原点は (0, 0, 0)
  • Rotation（回転）：各軸まわりの角度（度数）
  • Scale（拡縮）：各軸方向のサイズ倍率。初期値は (1, 1, 1)

Inspector の数値フィールドに直接入力するか、Scene ビューのギズモ（矢印・弧）をドラッグして変更できます。

⚠ Rotation の数値が「Euler 角」であることに注意。90° ずつ変えるとわかりやすい。
Section 03

保存しよう

プロジェクト保存とシーン保存

保存の種類

プロジェクト保存 ＆ シーン保存

1
シーンを保存する（重要！）Ctrl + S（Windows） / Cmd + S（Mac）で現在のシーンを上書き保存。初回は保存先を選ぶダイアログが出る（Assets フォルダ内に Scenes フォルダを作って保存）
2
プロジェクト全体を保存するメニュー File → Save Project。スクリプトや設定ファイルなど、シーン以外の変更も保存される
3
こまめに保存する習慣をUnity はクラッシュすると保存前の変更が消えます。作業の節目に必ず Ctrl/Cmd+S を！
⚠ Hierarchy のオブジェクト名の横に「*」が付いていたら未保存のサインです。
まとめ

今日学んだこと

  • Unityエディターは Scene / Game / Hierarchy / Inspector / Project の5エリアで構成される
  • Scene ビューで配置・視点操作、Game ビューでゲーム映像を確認する
  • Hierarchy でオブジェクトを管理し、Inspector で詳細設定を行う
  • 右クリック → 3D Object で Cube などを簡単に追加できる
  • Transform の Position / Rotation / Scale でオブジェクトの形を決める
  • Ctrl/Cmd+S でシーンをこまめに保存する
