Section 01

Prefabとは何か

再利用可能なオブジェクトの設計図

Prefab の概念

「設計図」からオブジェクトを量産する

  • Prefab ＝ オブジェクトの「テンプレート（設計図）」
  • 形・コンポーネント・スクリプト設定をまるごとファイルとして保存できる
  • Prefab からシーンに配置したオブジェクトを「インスタンス」と呼ぶ
  • 1つの設計図から何個でもインスタンスを作れる

Prefab がないと…

同じ障害物を10個置きたいとき、1個ずつ手動で作成・設定が必要。変更するときも1個ずつ…

Prefab があると…

設計図を1回作るだけ。あとは何個でも配置できる。変更も設計図1カ所を直すだけでOK！

Prefab のメリット

一括変更と量産ができる

  • 一括変更：Prefab を編集すると、シーン上のすべてのインスタンスに即座に反映される
  • 量産が簡単：ドラッグするだけで何個でも置ける。コピー＆ペーストよりも管理しやすい
  • スクリプトでの生成も可能：後の回で「ランダムに敵を出現させる」などの実装に使う
  • チーム作業にも有効：他のメンバーに Prefab ファイルを渡すだけでオブジェクトを共有できる

Prefab はプロのゲーム開発でも必須の概念。早いうちに使い方を身につけておこう！

Section 02

Prefabを作ってみよう

作成・配置・編集の流れ

Prefab 作成

HierarchyオブジェクトをProjectにドラッグ

1
シーンにオブジェクトを用意する例：Cube を追加してスケールを (2, 0.5, 2) にした「障害物」を作る
2
Project に「Prefabs」フォルダを作成するProject 右クリック → Create → Folder → 名前を「Prefabs」に
3
Hierarchy のオブジェクトを Prefabs フォルダにドラッグProject に青いキューブアイコンのファイルが作成されれば成功！
4
Hierarchy のオブジェクトが青くなる青い文字のオブジェクト ＝ Prefabのインスタンスのサイン
インスタンス化

Prefab をシーンに配置する

  • Project の Prefab ファイルを Scene ビューや Hierarchy にドラッグ するだけ
  • 何度でも繰り返すことで同じオブジェクトを複数配置できる
  • 各インスタンスの位置・回転は独立して変更できる
  • Hierarchy 上では名前の後ろに「(Clone)」は付かない（Prefab から直接配置した場合）

10個の障害物を並べたい場合、Prefab を10回ドラッグするだけ。あとは位置を調整すれば完成！

⚠ インスタンスを個別に編集すると、その変更は Prefab には反映されません（インスタンス固有の変更になる）。
Prefab の編集

全インスタンスに変更を反映させる

1
Project の Prefab ファイルをダブルクリック「Prefab 編集モード」に入る（Scene ビューがグレーになる）
2
この状態で変更を加える例：Scale や色（マテリアル）を変更する
3
Scene ビュー上部の「＜ Back」ボタンで戻る変更がすべてのインスタンスに反映される！

Prefab 編集モードで変えた内容はシーン上の全インスタンスに自動適用されます。これが Prefab の最大の強みです。

Section 03

簡単なステージを作ろう

地面・壁・障害物の配置と整理

ステージ構成

地面・壁・障害物の配置

地面（Floor）

Plane を追加。Scale を (5, 1, 5) 程度に広げる。プレイヤーが乗る床になる。

壁（Wall）

Cube を追加して細長く変形（例：Scale 0.2, 2, 10）。ステージの外壁になる。

障害物（Obstacle）

Cube を Prefab 化してから複数配置。回転・スケールを少し変えると見た目にバリエーションが出る。

配置のコツ

Scene ビューで真上から見る（テンキー 7）と配置しやすい。グリッドスナップ（Ctrl+ドラッグ）も活用しよう。

Hierarchy の整理

Empty Object で階層構造を作る

1
Hierarchy 右クリック → Create Empty空のゲームオブジェクトを作る。位置は (0,0,0) にリセットしておく
2
名前を付けて「フォルダ」代わりに使う例：「Stage」「Walls」「Obstacles」
3
オブジェクトをその下にドラッグ関連するオブジェクトをまとめて親子関係にする

Hierarchy が整理されると、どこに何があるかが一目瞭然になります。大きなプロジェクトほど重要！

マテリアル

オブジェクトに色を付ける

1
Project に「Materials」フォルダを作成アセットを整理する習慣を付ける
2
フォルダ内で右クリック → Create → Material分かりやすい名前を付ける（例：ObstacleMat）
3
Inspector の「Albedo」横のカラーボックスをクリックカラーピッカーで好きな色を選ぶ
4
Material を Hierarchy のオブジェクトにドラッグオブジェクトの色が変わる！Prefab にドラッグすれば全インスタンスに適用。
練習課題

Prefab で障害物を並べたステージを作ろう

1
Plane で地面を作るScale を調整して広めの地面を用意する
2
障害物の Cube を Prefab 化するマテリアルで赤や橙など目立つ色を付ける
3
Prefab を5個以上配置してコースを作る位置・角度・スケールを変えてバリエーションを出す
4
Hierarchy を Empty Object で整理する「Stage」「Obstacles」などのグループに分ける
まとめ

今日学んだこと

  • Prefab ＝ オブジェクトの設計図。一度作れば何度でも使い回せる
  • Hierarchy のオブジェクトを Project にドラッグして Prefab を作成する
  • Prefab をシーンにドラッグするだけでインスタンスを量産できる
  • Prefab 編集モードで変更するとすべてのインスタンスに反映される
  • Create Empty で親オブジェクトを作り、Hierarchy を整理する
  • Material を作成してドラッグすることでオブジェクトに色を付けられる
